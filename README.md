# 🚀 CryptoWatch: Real-Time Market Sentiment & Prediction Tracker

## 📌 Project Overview
It demonstrates a full data pipeline: from fetching live market data via REST APIs to performing trend analysis and predictive modeling using Scikit-Learn. It bridges the gap between traditional backend development and modern AI/ML workflows.

## 🛠️ Tech Stack & Skills
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Scikit-Learn (Linear Regression)
* **Data Sources:** Coinbase API (Price), Alternative.me API (Fear & Greed Index)
* **Environment:** Jupyter Notebook (.ipynb)

## 🧩 The "Smart Security" Analogy
> * **Python (The Wiring):** The core electrical system that connects every sensor and power source.
> * **APIs (The Cameras):** The lenses pointed at the market (Coinbase) to stream live data into our hub.
> * **Pandas (The Filing Cabinet):** Organizes thousands of raw images into neat, searchable folders (DataFrames).
> * **NumPy (The Calculator):** Performs instant, high-speed math to find averages and "Market Volatility."
> * **Matplotlib (The Monitor):** The big screen in the control room that turns numbers into visual price trends.
> * **Scikit-Learn (The Forecaster):** Analyzes past patterns to predict if it will "rain" (price drop) in the next hour.
> * **PyTorch (The Brain):** An artificial neural network that learns complex behaviors just like a human guard.

## 🧠 Key Features
1.  **Automated Ingestion:** Real-time polling of Bitcoin price and market sentiment.
2.  **Data Analysis:** Using **Pandas** to structure time-series data and **NumPy** for volatility calculations.
3.  **Predictive Modeling:** A **Scikit-Learn Linear Regression** model that forecasts the next price point based on session trends.
4.  **AI Insights:** A heuristic logic layer that provides automated trading advice (Bullish/Bearish/Flat) based on data-driven signals.

## 📈 Results
The tracker outputs a live visualization showing:
* Actual captured price points.
* A machine-learning-predicted target for the next time interval.
* An AI Analyst summary designed for professional decision-making.
